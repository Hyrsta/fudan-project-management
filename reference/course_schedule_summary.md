# Course Schedule and Key Milestones

## What this project is

This is a semester-long project management course project, not just a single coding task.
It combines project management artifacts, AI-assisted product/system work, stage demos, and a final roadshow.
The work is split across a project manager role, Team A, and Team B.

## Overall structure

- Weeks 4 to 6: project setup and first build phase
- Weeks 7 to 8: first roadshow
- Weeks 9 to 10: second build phase with risk, stakeholder, method, and ethics work
- Weeks 11 to 12: second roadshow
- Weeks 13 to 17: simulation run, audit/closure, and final defense

## My current understanding

The project has three layers at the same time:

1. **Project management layer**
   - kickoff, scope, schedule, governance, risk, stakeholder management, audit, closure

2. **Product / system layer**
   - MVP prototype, AI agent architecture, WBS assistant, risk warning agent, integrated AI agent system

3. **Course assessment layer**
   - two roadshows, one simulation run, one closure report, and final defense
